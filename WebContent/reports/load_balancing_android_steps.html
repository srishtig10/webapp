load_balancing_android_steps

Steps-
1 Create a new instance and install Apache2 on it
2 Then modified the Config file 000-default.conf in Sites Enabled Folder in Apache2
3 Created a Proxy Balancer member for Master and Slave and set up the Sticky Session
4 Two new instances for Master - Slave were created and SQL installed on them
5 Followed all stops for Master-Slave replication
6 Then we created an Android app using Android Studio. We used WebView feature in activity_main.xml and added code for onCreate() activity method in MainActivity.java file of the App
7 Created a new java class for easy maneouvering over the App
